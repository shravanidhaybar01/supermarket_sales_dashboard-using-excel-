📊 Supermarket Sales Dashboard using Excel

Overview

This repository contains our first group mini project completed during the Data Analytics Internship Program at Imarticus Learning.

The project focuses on analyzing supermarket sales data and creating an interactive dashboard in Microsoft Excel to transform raw data into meaningful business insights for decision-making.

Objectives

- Clean and organize raw sales data.
- Analyze sales performance using Excel.
- Build an interactive dashboard.
- Identify key business trends and insights.

Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning & Data Analysis
